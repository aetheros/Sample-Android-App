# TechFieldTool
An Android application that allows field technicians to connect to devices over wifi and perform a oneM2M node Retrieve.

## Support Android SDK Versions.
* Min: 23
* Target: 28

## Gradle
* Android plugin v3.6.3
* Gradle 5.6.4

## Java
* openjdk-11

## Dependencies
* AOSClient v1.0.0
* californium-core v2.2.2
* gson v2.8.6
* json v20200518
* zxing v3.3.0
